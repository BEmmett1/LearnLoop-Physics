# Software Development Life Cycle (SDLC)

## Process Model
Agile with iterative SDLC loops.

---

## Phases

### 1. Requirements
- Define MVP scope
- Lock non-goals

### 2. Design
- Architecture diagrams
- Data models
- Algorithm definitions

### 3. Implementation
- Build vertical slices
- Feature branches per slice
- Frequent commits

### 4. Testing
- Unit and integration tests
- Manual validation

### 5. Deployment
- Production deployment on Vercel
- Database migrations applied

### 6. Maintenance
- Monitor usage and drop-off
- Iterate based on feedback

---

## Sprint Rhythm
Each sprint repeats all SDLC phases on a small scale.
